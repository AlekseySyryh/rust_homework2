# Клиентская часть

Usage: client.exe [OPTIONS] --server <server> --udp <udp> --tickers <tickers>

Options:
      --server <server>              Адрес сервера
      --udp <udp>                    UDP адрес приема котировок
      --timeout <ping_timeout_secs>  Таймаут пинга в секундах [default: 5]
      --tickers <tickers>            Имя файла с тикерами

Клиент посылает на сервер команду, после чего начинает слушать котировки. Если пришла хоть одна котировка, на этот адрес начинают отправляться пинги с заданной периодичностью.

Если установлена системная переменная RUST_LOG=[уровень] то в консоль будут выводиться отладочные сообщения соответствующего уровня
