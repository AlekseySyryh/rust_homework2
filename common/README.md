# Общая бибилотека для котировок

Содержит структуру котировки и методы для сериализации/десериализации

Котровка содержит строку из 4 полей (ticker,price,volume,timestamp_ms) раздленных вертикальной чертой. 

Пробелы внутри полей не допускаются

Одна котировка на одну UDP-датаграмму

Список возможных тикеров определен внутри библиотеки.
