# Серверная часть

Usage: server.exe [OPTIONS]

Options:
      --port <port>                  Прослушиваемый порт [default: 8080]
      --timeout <ping_timeout_secs>  Таймаут пинга в секундах [default: 10]
      --generate-ms <generate_ms>    Частота генерации котировок в мс [default: 500]

Сервер слушает порт указанный в настройках. При получении команды 
```
STREAM [UDP] [TICKERS]\n
```
Начинает посылку котировок по тикерам на указанный адрес.

Ожидается что раз в timeout секунд от получателя будет приходить пакет PING. Если этого не произошло посылка котировок прекращается.

Если установлена системная переменная RUST_LOG=[уровень] то в консоль будут выводиться отладочные сообщения соответствующего уровня
